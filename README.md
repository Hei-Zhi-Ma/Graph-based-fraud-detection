# Graph-based-fraud-detection
一个基于图神经网络与深度子空间聚类的金融欺诈团伙识别系统，能够从复杂的用户交易网络中自动发现潜在的欺诈团伙
🔍 无监督学习 · 💰 金融风控 · 🕸️ 图神经网络 · 🎯 欺诈检测
🎯 项目亮点
渐进式架构：从简单MVP开始，逐步集成先进模块，确保每一步都可验证、可复现

小样本引导：仅需少量已知欺诈样本，即可大幅提升检测精度

强抗噪能力：通过图去噪和对比学习，在噪声数据下保持稳定性能

多视图融合：综合利用设备共享、地址共享、交易行为等多种关系

可解释性：提供团伙行为的可视化分析和关键特征解读

🚀 快速开始
环境要求
Python 3.8+

PyTorch 1.9+

PyTorch Geometric 2.0+

scikit-learn, pandas, numpy, matplotlib
🛠️ 核心模块
1. 图质量增强（抗噪基础）
CRTC：图结构补全，预测缺失的重要关系

GPNS：图去噪，自动识别并过滤噪声边

2. 智能表示学习
GraphSAGE + DGI：无监督图表示学习，捕捉节点结构特征

分层注意力：节点级和视图级注意力机制

3. 异常检测与验证
深度子空间聚类：在高维空间中发现低维团伙结构

AGCL引导：用已知欺诈样本指导聚类过程

多尺度检测：结合聚类、孤立点、社区发现

📈 开发路线
阶段一：MVP基础版（已完成）
基础图构建 + 简单GNN + KMeans聚类

目标：验证核心思路可行性

阶段二：增强抗噪性（进行中）
图去噪模块 + 鲁棒损失函数

目标：提升模型在噪声数据下的稳定性

阶段三：小样本引导（规划中）
已知欺诈种子引导 + 约束聚类

目标：利用少量先验知识大幅提升精度

阶段四：多视图融合（规划中）
多关系融合 + 注意力机制

目标：综合利用多种信息源
